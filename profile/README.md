# Ariadne

**Leave a thread. Find your way.**

---

Ariadne is an open-source organization building small, honest tools for the moment *after* pair programming ends — when the chat closes and you're alone with the code again.

Named after the mythological figure who gave Theseus a thread to navigate the labyrinth, we leave threads behind: clearer diffs, gentler errors, maps into unfamiliar codebases, patterns you can run and learn from.

### Principles

- **Offline-first** — no API key required for the default path
- **Educational** — tools should teach, not just automate
- **Transparent** — by **Ariadne**, hosted by **Pablo** · ariadne@pablovallejo.dev
